Learn web development. Build awesome things.

You quit your job, you told your friends and family you're doing something crazy, maybe you moved here. You're ready to learn, intensively, for the next twelve weeks. By the time you're done, you'll be able to build pretty much anything you see on the internet.

Prework

Get the foundations under your belt so we can hit the ground running. We'll point you in the right direction to take all the confusion out of learning on your own. Don't worry, we'll keep building on these skills during the course and teach you best practices.

  • Command Line: Give your computer instructions like a real programmer
  • HTML: Learn the basic building blocks of the front-facing web
  • CSS: Start understanding separation of concerns and building beautiful web sites with CSS
  • Git: Version control is a lifesaver
  • Basic functional Python: Write your first Python programs with variables, loops, functions, and conditional statements
  • Project: Build your personal website: Give your friends (and yourself) a taste of what you'll be working on by building your personal website. This will eventually turn into the portfolio site you use to show off your future projects
  • Installfest: Come meet your instructors and fellow bootcampers and make sure your computer is set up for the next twelve weeks

Object-oriented Python and Beginner Django

  • Object-oriented Python: Solidify the Python concepts you learned during your prework. Learn about object-oriented vs. functional programming, instantiating new objects, and debugging your Python code.
  • Intermediate Python: Become a Pythonista. Learn about inheritance, mixins, metaclasses, and decorators
  • Relational databases and introductory SQL: Understand the value of relational dataabases and learn basic SQL
  • Django ORM: Combine your knowledge of Python classes and databases to build relational models and manipulate a database with the Django ORM (object-relational mapper)
  • Beginner Django: Learn about MVC (model-view-controller), the software pattern behind Django, Rails, and much of modern web development. Build applications that have multiple models and use Django's advanced templating system. Learn how user authentication works and how to include it in your apps.
  • Project: Build a Django application with user authentication and database storage

Integrating JavaScript and Design

  • JavaScript and jQuery: Leverage your Python programming knowledge and learn JavaScript syntax. Manipulate the DOM with pure JavaScript. Use jQuery to supercharge your JavaScript, add animations, and build interactive web pages.
  • AJAX, JSON, and XML: Send asynchronous server requests to make your sites faster and bring in content from third-party applications
  • Integrating JavaScript with your Django applications: Take your Django apps to the next level with interactive JavaScript
  • Using third-party APIs: Query external services like Google and Instagram
  • Responsive design principles and using Bootstrap: Learn basic design principles to give you apps polish. Understand grids and the responsive patterns that allow a single site to live on desktop, tablet, and mobile. Learn Bootstrap, the responsive library that powers much of the internet.
  • Project: Integrate front-end technologies for a more interactive Django application

Prove you can program.

First things first: let's make sure everyone is comfortable with the Python prework you did. Instead of a boring review lecture, try out these challenge problems. They range from easy to difficult and will test you on variables, loops, data structures, conditionals, and functions. They can be pretty fun, and they are similar to questions you might be asked in an interview. These types of problems are often referred to as "FizzBuzz", after one of the best known ones (you'll know it when you see it).

Do not look up solutions online! But feel free to use the internet to refresh yourself on Python syntax or built-in methods. Don't let yourself get stuck on a single question. See how far you can get with these:

  • Coding Bat
  • Python Challenge
  • Coderbyte
